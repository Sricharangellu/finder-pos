# Module marketplace

## What is the module marketplace?

Ascend ships with 60+ modules across 14 verticals. Your base bundle activates the modules most relevant to your business type. The Module Marketplace lets you toggle individual modules on or off beyond the base bundle.

Location: **Settings → Module Marketplace**

## Base bundles

When you complete onboarding and choose a business type, a bundle of modules activates automatically:

| Business type | Base bundle includes |
|---|---|
| Retail | Catalog, Inventory, POS, Discounts, Loyalty, Gift Cards, Customers |
| Restaurant | Above + Tables, Kitchen Display, Bar Tabs, Reservations, Menu Modifiers |
| Hospitality | Above (retail base) + Room Billing, Guest Accounts, Housekeeping, Events |
| Services | Retail base + Appointments, Service Orders, Memberships, Commission |
| Healthcare | Retail base + Prescriptions, Patient Records, Insurance, Expiry Tracking |
| Automotive | Retail base + Vehicle History, Work Orders, Parts Catalog, Inspection |
| Manufacturing | Retail base + Production Orders, Bill of Materials, Raw Materials, QC |
| Rental | Retail base + Rental Contracts, Deposits, Asset Tracking |
| Entertainment | Retail base + Tickets, Access Control, Concessions, Season Passes |
| Education | Retail base + Fee Collection, Student Accounts, Enrollment |
| Golf | Retail base + Tee Sheet, Bookings, Memberships, Pro Shop |
| B2B / Wholesale | Retail base + Sales Orders, Purchasing, AP/AR, Quotes |
| E-Commerce | Retail base + Online Store, Fulfillment, Marketplace Sync, Shipping |

## Toggling modules

1. Go to **Settings → Module Marketplace**
2. Use the left sidebar to filter by vertical, or search by module name
3. Toggle a module on or off
4. The **Save changes** bar appears at the top when you have unsaved changes
5. Click **Save changes** — the sidebar navigation updates immediately

Core modules (marked with a lock icon) cannot be disabled — they are required for basic operation.

## Effect on navigation

When a module is enabled, its pages appear in the left sidebar automatically. When disabled, the pages are hidden (but the data is not deleted). Re-enabling the module restores all data and navigation.

## Common module combinations

**Full restaurant** — Tables + Kitchen Display + Bar Tabs + Reservations + Modifiers + Loyalty

**Appointment-based services** — Appointments + Service Orders + Memberships + Commission + Loyalty

**Multi-location retail** — Outlets + Warehouse Locations + Inventory Transfer + Multi-currency

**Compliance-heavy retail** — Age Verification + Tobacco Compliance + MSA Reporting + PACT Act

## Disabling a module

Disabling a module hides its UI but does not delete data. If you re-enable it later, all records are restored. If you want to permanently remove data from a disabled module, contact support.
